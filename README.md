# Calculadora

## Criar o projeto

Execute o comando abaixo, onde calculadora, é o nome do projeto:

```shell
$ create-react-app calculadora
```

Caso o create-react-app não esteja disponível, instale-o com o seguinte comando:

```shell
$ npm i -g create-react-app
```
